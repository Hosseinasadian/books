A simple **programming book library** built with **React**. Browse books, view details, chapters, and audio files. Supports light/dark theme and is deployed on **GitHub Pages**.

---

## Features

- List all books with cover, author, year, and description  
- View book details and chapters  
- Audio support for chapters  
- Light/Dark theme toggle  
- Responsive design  
- Client-side routing with **HashRouter** to prevent 404 on refresh

---

## Tech Stack

- React 19  
- React Router 7  
- GitHub Pages for hosting  
- Fetch API for data
